Mean, Median, and Mode with Examples

In statistics, mean, median, and mode are three important measures of central tendency that help us understand and describe data sets. Let's explore each of these concepts with examples:

Mean (Average)

The mean, often referred to as the average, is calculated by summing all the values in a data set and dividing by the number of values. Here's an example: 

Data set: 12, 15, 18, 22, 25

Calculating the mean:

Mean = (12 + 15 + 18 + 22 + 25) / 5 = 92 / 5 = 18.4

So, the mean of this data set is 18.4.

Median

The median is the middle value in a data set when it is ordered from least to greatest. If there is an even number of values, the median is the average of the two middle values. Here's an example: 

Data set: 14, 18, 21, 25

Arranging the data in ascending order:

14, 18, 21, 25

Since there is an even number of values (4), the median is the average of the two middle values (18 and 21): 

Median = (18 + 21) / 2 = 39 / 2 = 19.5

So, the median of this data set is 19.5.

Mode

The mode is the value that occurs most frequently in a data set. Here's an example: 

Data set: 12, 15, 15, 18, 22, 25, 25, 25

In this data set, the mode is 25 because it occurs more frequently (three times) than any other value.
