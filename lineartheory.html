Linear Search Algorithm

In this article, we will explore the Linear Search Algorithm, a simple and straightforward searching technique used to find a specific element in a list or array. Linear search is a basic yet essential algorithm, especially when you need to locate an item in an unsorted collection.

Linear Search is preferred in scenarios where:

  • Efficiency and speed are not the primary concerns.
  • The list or array is small or unsorted.

        Algorithm
        Linear Search, also known as Sequential Search, is a simple searching algorithm that scans each element in a list or array one by one until it finds the target element or reaches the end of the collection. Here's a high-level summary of the algorithm:

        begin LinearSearch(arr, target)  
           for each element in arr
               if element equals target
                   return the index of the element
               end if
           end for
           return -1  // Element not found
        end LinearSearch

    Working of Linear Search Algorithm

     
        Linear Search is a straightforward algorithm that works as follows:

        1. **Iterate Through Elements**: Start from the beginning of the list or array and examine each element one by one, in order.

        2. **Comparison**: Compare the current element with the target element that you are searching for.

        3. **Element Found**: If the current element matches the target, return the index of that element.

        4. **Element Not Found**: If you reach the end of the list or array without finding the target, return -1 to indicate that the element is not present.

        Linear Search is easy to understand and implement, making it a suitable choice for small collections or situations where sorting the data is not feasible or necessary.
